# Level 03｜新月岩台·安置营地

## 概念探索与取舍

| 方案 | 核心幻想 | 主动作 | 地图结构 | 数学结合 | 主要成本 |
|---|---|---|---|---|---|
| A 新月岩台营地（采用） | 读懂风、雨和岩壁后与伙伴把荒台变成温暖住所 | 比较选址、拉尺、搬卷垫、展开压平、合作立帐 | 新月山脊＋A/B候选地＋6×4背风台＋回访绳桥 | 实测总面积，扣2×2火塘，以10块2×1垫真实铺满 | 主线较长，需坡地步态、精确槽位、帐篷多阶段状态 |
| B 巨榕树屋营地（淘汰） | 在树冠间搭建离地小屋 | 攀梯、装平台、拉绳桥 | 中央巨树＋三层环形枝台 | 用平台面积扣树干截面，再算木板数 | 需要垂直导航、防坠落和遮挡处理；方形扣除在树干上也显得人为 |
| C 风暴沙滩营地（淘汰） | 在潮水和强风前搭起海滩避难所 | 插桩、堆沙袋、拉防水布 | 长弧沙滩＋潮线推进＋沙丘背风窝 | 按安全区面积与帐篷占地求防潮垫数 | 倒计时与动态潮线会给儿童压力，且重复第一关海岸视觉语言 |

- 选择A：选址知识能由飘带、积水、落石缓冲直接观察；面积答案随后对应10个不覆盖火塘的精确槽，最后帐篷、灯与伙伴行为把数学结果变成可生活空间。
- 淘汰B/C：B的垂直系统和树冠镜头成本过高，C的限时压力及海岸重复削弱第三关应有的沉稳建设高点。
- **去掉题目弹窗仍成立的乐趣**：比较两块地、与刺猬拉尺、把十卷草垫铺成完整地面并共同撑起亮灯帐篷仍足以成立；情绪由高处风声中的谨慎→选址确信→重复建造的专注→围火安定与归属感。

## 设计判断

“靠岩石”不是一个孤立道具：岩壁要造成背风区、雨水流向和视线遮挡，玩家通过飘带、积水、松针分布比较两个候选地，先理解为何选址，再测量铺垫。营地完成后要从空台变为可生活的场所。设计采用官方开发者访谈中“所有元素围绕同一主题、玩家亲手创建”的原则，以及《动物森友会》官方呈现的制作/布置/与岛民共享生活的活动结构，但不复制关卡或视觉。[塞尔达官方访谈](https://www.nintendo.com/us/whatsnew/ask-the-developer-vol-9-the-legend-of-zelda-tears-of-the-kingdom-part-2/)｜[动物森友会官方产品页](https://www.nintendo.com/us/store/products/animal-crossing-new-horizons-switch/)

## 1. 地图、选址与视线

`32×24m` 新月形岩岛，西北弧形山脊 `y=1.8–3.2m`，中部鞍部 `y=0.8m`，东南营地台 `y=1.0m`，码头 `y=0`。不是圆岛，也不是平地放一块岩石。

```text
北 +Z       山脊观风台(-5,9)──海鸟巢(2,10)
          ╭挡风岩壁╮  候选A湿洼(1,5)[积水/落石]
西 -X 码头(-14,0)─鞍部路径─候选B台地(5,1)：6×4铺设区──草垫棚(13,1) 东
                         火塘2×2     帐篷基座(5,5)
南 -Z       回访捷径/绳桥──────────观星崖(8,-8)
```

- 抵达时橙色勘测旗位于高处，吸引玩家上鞍部；从观风台可同时看到积水候选A和岩壁背后的干燥候选B。
- 环境逻辑：B距岩壁3m避开落石线，入口朝东南；地面向南2%排水；火塘距帐篷/草木≥2m；草垫棚贴干燥东壁但不挡风口。
- 碰撞：山脊不可攀段用碎石+陡坡双提示；落石缓冲区以浅碎屑扇形标出；营地上空/四周2m保持镜头净空。
- 返回：西码头通第2岛，南绳桥解锁后直达第1岛的潮池侧；所有铺设中间态保留。

### 施工坐标定义（均使用 `Vector3(x,y,z)`，1单位=1米）

- 候选B铺设矩形：地面 `y=1.0`，边界 `x[2,8]、z[-1,3]`，长边沿X轴6m、短边沿Z轴4m；四角为 `(2,1,-1)`、`(8,1,-1)`、`(8,1,3)`、`(2,1,3)`。
- 火塘：中心 `Vector3(5,1.04,1)`，不可铺边界 `x[4,6]、z[0,2]`，正好2×2m。
- 10块2×1m草垫（长边沿X轴）的中心坐标：南排 `(3,1.06,-0.5)`、`(5,1.06,-0.5)`、`(7,1.06,-0.5)`；中南排 `(3,1.06,0.5)`、`(7,1.06,0.5)`；中北排 `(3,1.06,1.5)`、`(7,1.06,1.5)`；北排 `(3,1.06,2.5)`、`(5,1.06,2.5)`、`(7,1.06,2.5)`。省略的 `(5,1.06,0.5)` 与 `(5,1.06,1.5)` 两块恰好组成2×2m火塘。
- 帐篷占地：`x[3.5,6.5]、z[4,5.5]、y=1.0`，入口中心 `Vector3(5,1,4)` 朝南；距火塘北边 `z=2` 为2m净空。
- 挡风岩壁脚线：从 `Vector3(-1,1.8,9)` 经 `(4,2.4,8.5)` 至 `(9,2.0,8)`；不可进入落石缓冲带为 `x[0,9]、z[6.5,9]`。帐篷北边 `z=5.5` 与缓冲带南边 `z=6.5` 尚有1m净带，不重叠。
- 候选A湿洼：`x[-1,3]、z[4.5,7]`，部分落入缓冲带且排水沟终点积水，因此判为不合格；候选B及帐篷均不进入A。

## 3. 非数学微活动

1. **比较候选地｜主线必做**：在A/B各检查飘带、地面和岩壁距离，共6个观察点；玩家把三枚图标卡（避风/排水/落石）贴到正确地点。没有算式，选错由飘带/水迹动画反证。目的：让“靠岩壁但留缓冲”成为理解后的选址；完成才把勘测旗移到B并进入测量。
2. **帮助海鸟修巢｜可选**：强风吹落3根软草，位于主路旁；拾起后放在巢下篮，海鸟叼走并在巢边加一圈草。目的：让风再次参与玩法，奖励蓝羽；不影响选址、测量或通关。

## 4. 测量与数学

玩家与刺猬共同拉卷尺：长边6m（角色倒退6个1m标记，脚步逐点停）、短边4m、火塘2×2m；草垫实体标签2㎡/块。所有数据先在世界出现。

- 题面：营地6×4m，中央2×2m火塘不铺，每块草垫覆盖2㎡，需要几块？
- 解法：`6×4=24㎡`，`2×2=4㎡`，`(24-4)÷2=10块`。
- 验算：10块按5列×2排覆盖20㎡，加火塘4㎡等于24㎡。
- 选项8：虚铺后留4㎡斜线缺口；10正确；12：会盖住火塘，火塘石圈闪动提示不可铺。
- 来源：`original_scene`；卷尺/火塘/垫牌；`project_original`；2026-06-21。
- 难度：概念2、步骤2、信息1、表征2、计算1，总8=C；课程批准。

## 5. 亲手建营、角色动画和验证

1. 答对只出现第一个轮廓。每块：棚边抱卷垫（0.35s）→负重走到指定2×1槽→放下（0.2s）→双手向前滚开（0.55s）→脚踩两角压平（左右各0.18s）→绳牌翻面为绿色。共10块。
2. 空手移动：上坡身体前倾不超过6°、步幅变短；下坡手臂外展保持平衡；岩面脚步硬、草垫脚步闷。背部必须保持挺直圆润，不允许用驼背表达负重。
3. 表情：测量时嘴微张专注并看卷尺端点；展开草垫时舌尖轻露/眉内收；对齐成功闭眼笑0.35s；靠近火塘错误槽时惊讶圆嘴并摇头。
4. 10块铺完并不立即结算：玩家与刺猬各扶一根帐篷杆，玩家连续操作2次立杆，再走到4个绳钉中任意2个拉紧（刺猬完成另2个）。帐篷从塌布变成完整体，门灯亮。
5. 最终验证：玩家绕营地一周检查排水口、火塘净空、帐篷绳；随后可点火，火苗/暖光/水壶蒸汽/刺猬坐垫出现，角色坐下揉脚并满足微笑。营地从施工空间转为生活空间。
6. 回访：可坐垫休息恢复表情、在观星崖看星光粒子；帐篷、火、10垫、鸟巢、飘带状态保留。

## 6. 状态与验收

主线：`ARRIVE → SITE_COMPARE_0..6 → SITE_CHOSEN → MEASURE_0..3 → PLAN → MAT_n → FLOOR_DONE → TENT_0..4 → SAFETY_CHECK_0..3 → CAMP_LIVE`。可选修巢独立为 `NEST_0..3 → NEST_REPAIRED`，主线不查询该状态。

- [ ] 新月形、多级岩脊、两候选地和南北环路均可从地形读出。
- [ ] 选址活动必须让风、排水、落石三项都有物理证据；修巢有鸟叼草反馈。
- [ ] 数据由卷尺共同动作获得；10块各有抱、负重走、滚开、压角、状态牌。
- [ ] 角色有坡地步态、负重但不驼背、测量/用力/成功/错误表情。
- [ ] 10块不得覆盖2×2火塘；每块占2㎡并形成可验算布局。
- [ ] 铺完还需合作立帐篷及安全巡检，完成营地出现灯、火、蒸汽、伙伴行为。
- [ ] 可往返三岛，任一中间状态都持久；自动验收覆盖离岛/回岛、错项和碰撞。

## 7. 相邻关差异与风险

- 对第1关：岩石高差/选址比较/大件展开，非海湾平地/钓鱼/线性桥。
- 对第2关：干燥新月山脊/候选地决策/合作建造，非低湿三叶/追水/推车运输。
- 风险：任务链比前两关长；候选地比较是主线必做，只有修巢可选。主线必须保留选址→测量→10垫→帐篷→3点巡检；若纵切片时长超标，先缩短搬运距离，不能自动铺垫、删除验证或删除选址的三类环境证据。

## 8. 交付分层

- **最小可交付（不得删减）**：新月岛和真实高差；上述施工边界/岩壁/缓冲带；A/B六点比较及三种环境反证；3次测量、数学题、精确坐标上的10次搬运展开、火塘禁铺；合作立帐篷、3点安全巡检；坡地/负重步态及四类表情；修巢可选活动完整反馈；三岛往返和所有中间态保存。
