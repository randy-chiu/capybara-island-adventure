# 卡皮巴拉海岛游戏工程

这是项目唯一的 Godot 4.7 游戏工程。所有玩法和关卡都在这里维护。

## 运行

```bash
cd Game
godot --editor project.godot
```

也可以直接运行：

```bash
godot --path Game
```

在 Finder 中也可以直接双击仓库根目录的 `试玩卡皮巴拉海岛.command`。

## 操作

- 方向键或 `WASD`：移动
- `空格` 或 `E`：互动
- 也可以点击右下角互动按钮

## 当前任务闭环

1. 自由探索并收集闪光鱼饵
2. 前往码头钓鱼
3. 根据红色光柱找到断桥
4. 解答修桥数学题
5. 根据金色光柱找到并打开宝箱
6. 获得星光贝壳并显示任务结算
7. 选择“进入下一关”，抵达新岛并寻找 3 处水滴线索
8. 跟随蓝色光柱找到淡水湖，完成第二关探索目标

## 当前实现重点

- 角色待机呼吸、走路步伐、手脚摆动、身体起伏和平滑转向
- 岛屿、水面、码头、树木摆动和动态目标光柱
- 明确的陆地、码头、断桥可行走边界
- 完整的“先游戏，后在关键目标处解题”流程
- 开始页、任务提示、物品状态、错误反馈、完成结算和重新开始

当前角色和环境仍是程序生成的低多边形占位模型。下一阶段应换成 Blender 制作或合法授权的原创模型、骨骼动画、PBR 材质和环境音效。
